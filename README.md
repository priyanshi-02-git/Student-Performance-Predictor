# STUDENTS PERFORMANCE PREDICTOR



## Project Description

The project is a simple Machine Learning application where the final marks of a student can be predicted based on various factors such as the hours of study, attendance, previous marks, hours of sleep, and assignment score.

The project is based on the concept of Linear Regression and also includes a simple Graphical User Interface application using Python's Tkinter library.


## Objective

### The objective of the project is to:

- Get a basic idea of Machine Learning concepts
- Implement Linear Regression on a real-time dataset
- Develop a simple GUI-based application
- Display the data in the form of a graph


## Technologies Used
- Python
- Pandas library for handling the dataset
- Scikit-learn library for the Machine Learning model
- Tkinter library for creating the GUI
- Matplotlib library for visualizing the data using graphs
- Pillow library for handling images

## Project Features

1. Easy to use graphics interface
2. Input student details 
3. It Uses ML model to calculate final marks estimated to help students see potential performance levels. (e.g. Excellent) 
4. It Will show a 24 hour distribution of this students lifestyle through a bar. 
5. It Will show a graph through Study hours vs Marks.
6. It Will include images to make it a better UI design.


### Input Parameters
- Study Hours
- Attendance
- Previous marks 
- Sleep Hours
- Assignment Score

 
### Output
- Prediction on final marks for students. 
- Prediction on performance category of student.
- Graphical data showing the relationship of study hour and the predicted resulting.



### Machine Learning Model
The model will use Linear Regression which is a supervised ML algorithm and learn to predict continuous values.


## Conclusion
This project is a demonstration of what can be accomplished with the use of Machine Learning such as predictive analysis of student performance and visuals combined with ML to create a better user interface.
